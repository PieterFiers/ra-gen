Waar haal ik mijn API-sleutel?

Ga naar console.anthropic.com → Settings → API Keys → Create Key.

De sleutel begint met sk-ant- en wordt veilig opgeslagen op dit toestel. Ze verlaat nooit dit apparaat, gaat rechtstreeks naar Anthropic.

Privacy & veiligheid

De sleutel wordt opgeslagen in de lokale opslag van je browser (localStorage). Ze staat niet in de code op GitHub en is alleen zichtbaar op dit toestel. Verwijder ze via de knop hieronder als je het toestel doorgeeft.
